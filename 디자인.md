# 디자인 명세 — job-notion-clipper

---

## 1. 디자인 방향

- **미니멀 + 정보 밀도**: 팝업 크기 제한(360×580px) 안에서 핵심 정보만 노출
- **상태 중심 UI**: 로딩 → 미리보기 → 저장완료 각 상태가 명확히 구분
- **Notion 색상 테마와 조화**: 흰 배경 + 회색 텍스트 + 포인트 컬러(블루)

---

## 2. 팝업 크기 & 레이아웃

```
┌─────────────────────────────┐  width: 360px
│  🗂 job clipper        [×]  │  header: 48px
├─────────────────────────────┤
│  [저장하기]  [최근 목록]    │  tab bar: 40px
├─────────────────────────────┤

│       (탭 콘텐츠 영역)      │  body: ~492px
│                             │
└─────────────────────────────┘  total: 580px
```

## 5. 컴포넌트 명세

### 5-1. Header

```
┌─────────────────────────────┐
│ 🗂  job clipper             │
└─────────────────────────────┘
```

- 배경: `white`, 하단 border: `border-b border-gray-200`
- 아이콘: 16px emoji 또는 SVG
- 높이: `h-12 px-4`

---

### 5-2. Tab Bar

```
┌──────────────┬──────────────┐
│  저장하기    │  최근 목록   │
│  (활성: 파란 │              │
│   하단 선)   │              │
└──────────────┴──────────────┘
```

- 탭 2개 균등 분할: `grid grid-cols-2`
- 활성 탭: `border-b-2 border-blue-600 text-blue-600 font-medium`
- 비활성 탭: `text-gray-500 hover:text-gray-700`
- 높이: `h-10`

---

### 5-3. SaveTab — 상태별 UI

#### 상태 A: 로딩 중

```
┌─────────────────────────────┐
│                             │
│        ◌ 공고 분석 중...    │
│        (스피너 + 텍스트)    │
│                             │
└─────────────────────────────┘
```

- 스피너: Tailwind `animate-spin` 원형
- 텍스트: `text-sm text-gray-500`
- 레이아웃: `flex flex-col items-center justify-center h-64 gap-3`

---

#### 상태 B: 미리보기 (파싱 완료)

```
┌─────────────────────────────┐
│  카카오                ✓URL │  회사명 + URL 링크
│  시니어 프론트엔드 개발자   │  포지션
├─────────────────────────────┤
│  기술스택                   │
│  [React] [TypeScript] [+2]  │  multi_select 배지
├─────────────────────────────┤
│  연봉          마감일       │
│  5천~8천만원   D-14         │  2열 그리드
├─────────────────────────────┤
│  한줄메모                   │
│  ┌─────────────────────┐   │
│  │ 메모를 입력하세요…  │   │  textarea
│  └─────────────────────┘   │
│                             │
│  [    Notion에 저장하기   ] │  primary 버튼
└─────────────────────────────┘
```

- 카드 전체: `bg-gray-50 rounded-lg p-4 mx-3 my-2`
- 기술스택 배지: `bg-blue-100 text-blue-700 text-xs px-2 py-0.5 rounded-full`
- textarea: `w-full border border-gray-200 rounded-md p-2 text-sm resize-none h-16`
- 저장 버튼: `w-full bg-blue-600 hover:bg-blue-700 text-white font-medium py-2.5 rounded-lg`

---

#### 상태 C: 중복 감지 배너

```
┌─────────────────────────────┐
│  ⚠️ 이미 저장된 공고예요    │  yellow 배너
│  [덮어쓰기]  [취소]         │
└─────────────────────────────┘
```

- 배너: `bg-yellow-50 border border-yellow-200 rounded-lg p-3 mx-3`
- 텍스트: `text-yellow-800 text-sm`
- 버튼 2개: `gap-2 flex` — 덮어쓰기(outline), 취소(ghost)

---

#### 상태 D: 저장 완료

```
┌─────────────────────────────┐
│                             │
│         ✅                  │
│    저장이 완료됐어요!        │
│    Notion에서 확인하기 →    │  링크
│                             │
└─────────────────────────────┘
```

- 아이콘: `text-green-500 text-3xl`
- 링크: `text-blue-600 underline text-sm`
- 2초 후 자동으로 미리보기 상태로 리셋

## 7. 반응형 / 접근성

- 팝업 고정 크기 (360px) — 반응형 불필요
- 버튼 최소 높이 `44px` (모바일 터치 기준 준수)
- 포커스 링: `focus:outline-none focus:ring-2 focus:ring-blue-500`
- 색상 대비: WCAG AA 기준 (`text-gray-900` on `white` = 21:1)
