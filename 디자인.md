## 레이아웃

```
┌──────────┬────────────────────────────────────────┐
│          │  Header: "공고 관리"  [+ 공고 추가]     │
│  사이드   ├────────────────────────────────────────┤
│  바       │  [전체 N]  [관심 N]  [지원완료 N]  [결과대기 N]  │
│ (gray-900)├────────────────────────────────────────┤
│          │  검색 인풋  |  상태 필터 드롭다운        │
│          ├────────────────────────────────────────┤
│          │  테이블: 회사명 / 포지션 / 상태 / D-Day / 메모 / 액션 │
└──────────┴────────────────────────────────────────┘
```

## 색상 시스템

| 역할 | 클래스 | 설명 |
|------|--------|------|
| 사이드바 배경 | `bg-gray-900` | 다크 네이비 |
| 메인 배경 | `bg-gray-50` | 라이트 그레이 |
| 카드/테이블 배경 | `bg-white` | 흰색 |
| 포인트 컬러 | `indigo-600` | 버튼·포커스 링 |
| 관심 | `blue-100 / blue-700` | |
| 지원예정 | `purple-100 / purple-700` | |
| 지원완료 | `green-100 / green-700` | |
| 결과대기 | `yellow-100 / yellow-700` | |

## D-day 색상 규칙

| 조건 | 색상 |
|------|------|
| 마감 지남 | `text-gray-400` (흐림) |
| 3일 이하 | `text-red-600 font-semibold` |
| 4~7일 | `text-orange-500 font-semibold` |
| 8일 이상 | `text-gray-600` |
| 마감일 없음 | `-` 표시 |

## 컴포넌트 명세

### Header
- 좌: 페이지 타이틀 "공고 관리"
- 우: `[+ 공고 추가]` 버튼 (indigo-600)

### StatsCard (4개)
- 전체 / 관심 / 지원완료 / 결과대기 카운트
- 흰색 카드, 둥근 모서리, 숫자 크게 표시

### JobTable
- 검색 + 상태 필터 상단 배치
- 컬럼: 회사명(링크) / 포지션 / 상태(드롭다운) / D-Day / 메모 / 액션(수정·삭제)
- 빈 상태: "등록된 공고가 없습니다" 안내

### JobFormModal
- 오버레이 모달 (480px 고정 너비)
- 상단: AI 공고 텍스트 분석 섹션 (indigo-50 배경, textarea + AI 분석 버튼)
- 필드: 회사명*, 포지션*, URL, 기술스택(쉼표 구분 + AI 추천 버튼), 마감일, 상태, 메모
- 모바일: 하단 시트(bottom sheet) 형태, 데스크탑: 중앙 모달

### CoverLetterModal
- 오버레이 모달 (600px 고정 너비)
- 강조할 경험 textarea → AI 자소서 생성 버튼
- 결과: 지원동기 / 직무역량 / 입사포부 섹션별 표시 + 개별/전체 복사
- 저장 버튼으로 해당 공고에 자소서 초안 연결
