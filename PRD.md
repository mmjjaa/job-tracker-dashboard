# Job Tracker Dashboard — PRD

## 목표

취업 준비생이 채용 공고를 등록하고 지원 현황을 한눈에 관리하는 웹 대시보드

---

## 핵심 기능 (MVP)

| # | 기능 | 설명 |
|---|------|------|
| F-01 | 공고 추가 | 회사명/포지션/URL/기술스택/마감일/메모 입력 모달 |
| F-02 | 상태 변경 | 관심 → 지원예정 → 지원완료 → 결과대기 인라인 변경 |
| F-03 | 대시보드 통계 | 상태별 카운트 카드 4개 (전체 / 관심 / 지원완료 / 결과대기) |
| F-04 | 목록 조회 | 테이블 뷰, 상태별 필터, 회사명·포지션 검색 |
| F-05 | 공고 수정/삭제 | 수정 모달 + 삭제 확인 다이얼로그 |
| F-06 | D-day 표시 | 마감일까지 남은 일수 색상 강조 |
| F-07 | 로컬 저장 | Zustand + localStorage, 새로고침 후에도 유지 |

---

## 제외 (v1)

- 서버/DB 연동
- 인증 (로그인)
- 공고 자동 파싱
- 엑셀 내보내기
- 칸반 보드 뷰

---

## 상태 정의

| 상태 | 색상 | 설명 |
|------|------|------|
| 관심 | blue | 저장해둔 공고 |
| 지원예정 | purple | 지원할 예정 |
| 지원완료 | green | 이미 지원함 |
| 결과대기 | yellow | 결과를 기다리는 중 |
