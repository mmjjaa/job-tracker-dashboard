# 데일리 채용 브리핑 스킬

## 시작

다음 파일을 읽어 사용자 프로필을 파악한다:
- `my-profile/job-preferences.md`
- `my-profile/experience-blocks.md`

그 다음 `data/jobs.json` 파일이 있으면 읽어서 현재 지원 현황을 파악한다.
파일이 없으면 사용자에게 안내한다:
> "대시보드(http://localhost:5175)에서 '내보내기' 버튼을 눌러 jobs.json을 저장해 주세요."

---

## 1단계 — 마감 임박 분류

오늘 날짜 기준으로 공고들을 분류한다:

**🚨 긴급 (오늘~내일 마감)**
→ 자소서 작성 여부 확인. 미작성이면 즉시 시작 권고.

**⚡ 이번 주 (2~7일 이내 마감)**
→ 오늘 시작 권장.

**📅 다음 주 이후**
→ 일반 관리.

---

## 2단계 — 새 공고 분석 (선택)

사용자가 새 공고 텍스트를 붙여넣으면:

`my-profile/job-preferences.md`의 기술 스택, 선호 조건과 대조해서 매칭 스코어를 산출한다.

| 회사 | 포지션 | 매칭 | 마감 | 추천 여부 |
|------|--------|------|------|-----------|
| (회사명) | (포지션) | 92% | MM/DD | ⭐ 적극 지원 |
| (회사명) | (포지션) | 74% | MM/DD | 🔵 고려 |
| (회사명) | (포지션) | 45% | MM/DD | ❌ 불일치 |

매칭 기준:
- 90% 이상 → ⭐ 적극 지원 (기술 스택 + 도메인 + 경력 모두 부합)
- 70~89% → 🔵 고려 (핵심 조건 부합, 일부 미달)
- 69% 이하 → ❌ 불일치 (제외 사유 설명)

---

## 3단계 — 브리핑 출력

아래 형식으로 오늘의 브리핑을 출력한다:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 채용 브리핑 — YYYY년 MM월 DD일
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚨 긴급 마감
  · [회사] [포지션] — MM/DD 마감
    → 자소서 상태: 작성중 / 미작성
    → 권고: 오늘 제출 목표

⚡ 이번 주 마감
  · [회사] [포지션] — MM/DD 마감

📅 다음 주
  · [회사] [포지션] — MM/DD 마감

🔍 신규 매칭 공고
  ⭐ [회사] [포지션] (MM/DD) — 매칭 NN%
  🔵 [회사] [포지션] (MM/DD) — 매칭 NN%

💡 오늘 할 일
  1. (긴급 공고 자소서 작성 또는 제출)
  2. (이번 주 마감 공고 시작)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

브리핑 출력 후 사용자에게 자소서 작성이 필요한 공고가 있는지 확인한다.
있으면 `/cover-letter` 스킬로 바로 연결 제안.
